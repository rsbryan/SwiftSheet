# Swiftsheet

Swiftsheet is a modern web application built with **Next.js**, **Supabase**, and **Stripe**, allowing users to upload, store, and display PDFs that can be linked to **NFC chips** via unique URLs. Designed for **restaurants and businesses**, it simplifies virtual menu hosting and document sharing.

## Features

- ✅ **User Authentication** – Secure account creation and login via Supabase Auth.
- 📂 **PDF Upload & Storage** – Upload, store, and manage PDF files effortlessly.
- 🔗 **NFC-Linked URLs** – Generate unique URLs for each PDF to pair with NFC chips.
- 🖥️ **Seamless Display** – PDFs are easily accessible through clean, mobile-friendly UI.
- 💳 **Stripe Integration** – Monetization features (subscriptions, one-time purchases) for premium users.

## Tech Stack

- **Frontend:** [Next.js](https://nextjs.org/), [TypeScript](https://www.typescriptlang.org/)
- **Backend:** [Supabase](https://supabase.com/) (PostgreSQL, Authentication, Storage)
- **Payments:** [Stripe](https://stripe.com/)
- **Styling:** Tailwind CSS
- **Deployment:** Vercel
